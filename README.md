# Pokedex

A Vue 3 + TypeScript Pokedex focused on form browsing, collection tracking, and trainer-friendly progress stats.

## Live Links

- Netlify: [the-pokedex-site.netlify.app](https://the-pokedex-site.netlify.app/)
- GitHub Pages: [luisdfilipe.github.io/Pokedex](https://luisdfilipe.github.io/Pokedex/)

## Current Features

- Browse Pokemon species and forms with sprite support
- Search by name, type, or number
- Filter by generation, base forms, boxable forms, megas, and gigantamax forms
- Track normal and shiny collection progress in local storage
- Review collection stats by generation and type
- Persistent filter settings between sessions

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Vue Router
- Phaser for the game prototype

## Local Setup

```bash
npm install
npm run dev
```
